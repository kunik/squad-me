# Compress / Log

1. Review the conversation, working tree, and relevant verification results.
2. Check `.agents/logs/` for a log from today on the same topic.
3. Update that log, or create `.agents/logs/YYYY-MM-DD-HHMM-<slug>.md`.
4. Include only sections with content: Summary, Key decisions, Files changed,
   Verification, and Pending.
5. State facts and outcomes, not a transcript. Never include secrets or tokens.
6. Confirm the path written.
