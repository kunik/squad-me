# Always-dark site footer

## Summary
Added a shared always-dark `SiteFooter` (mono lockup) on Home, auth, and
account chrome via `SiteChrome` / AccountShell. Short pages keep content at
≥100dvh so the footer stays below the fold. Wired `/privacy`, `/terms`,
`/contact` stub pages plus UA/EN footer strings and guest/auth-aware nav.

## Key decisions
- Footer surface is hard-coded dark (`#050609`), not `data-theme`.
- Mono PNG lockup (`logo-full-mono.png`) only in footer; theme brand chrome unchanged.
- Legal/contact routes are public stubs and onboarding-guard exempt.
- Copyright sits right-aligned in the footer bar.
