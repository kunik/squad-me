# Auth form exit link (AUTH-006)

## Summary
Auth forms under `AuthLayout` now share `AuthExitLink`: signed-in → `/profile`
(«До профілю»), guest → `/` («На головну»). Documented in repo + KB so new
forms keep the same footer. Also dropped lang/theme from auth cards (Home only).

## Key decisions
- One helper (`authExitTarget` / `AuthExitLink`), not per-page hardcodes.
- `/forgot-password` is dual-entry (profile + guest) — must follow session state.
- Hide `GuestUtilities` on auth forms; lang/theme stay on Home.
